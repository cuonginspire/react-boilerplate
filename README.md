> ⚛️🐣 React Boilerplate with Hooks, Redux, Redux-Saga and React-Router.

## Installation

```sh
$ npm i
# or
$ yarn
```

## Usage

```sh
$ npm start
# or
$ yarn start
```

## Build

```sh
$ npm run build
# or
$ yarn build
```
